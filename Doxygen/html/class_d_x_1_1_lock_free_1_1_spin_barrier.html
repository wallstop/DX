Detailed Description

SpinBarrier is designed for use to synchronize workflow between multiple threads. Calls to wait() will block until some specified number of threads have called wait(), after which all threads will unblock and continue execution.

Note
SpinBarrier is a form of a single-use barrier. For continued use, call reset() in a threadsafe fashion.
